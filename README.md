#API testing with Rest Assured.

API testing with the REST-Assured library in Java.

#Setup:
Download JDK-17 and set up the environment.
Install IntelliJ IDEA Community Edition.
Add the following dependencies to build.gradle from MAEVEN repository:
Test ng:https://mvnrepository.com/artifact/org.testng/testng
res-assured:https://mvnrepository.com/artifact/io.rest-assured/rest-assured
